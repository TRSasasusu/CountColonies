# CountColonies
生命情報実験Aの2番目の実験とかで菌のコロニーを数えるときに少し便利になるかもしれない，Windowsでしかたぶん動かないソフトウェア．  
## Usage
1. [ここ](https://github.com/TRSasasusu/CountColonies/releases/download/1.0.0/CountColonies.exe)からダウンロードする．
2. ダブルクリックして開く．
3. コロニーの写った写真をドラッグアンドドロップする．
4. ひたすらコロニーをクリックする．
5. 「コロニー数」を記録する．
## Caution
Windowsでしか動きません．
